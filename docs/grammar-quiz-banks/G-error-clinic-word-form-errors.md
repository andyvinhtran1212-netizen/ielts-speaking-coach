# ===== item_key 1 · Noun → Adjective (danh từ bị dùng nhầm thành tính từ) =====

---

headword: "wfe-noun-to-adjective"
skill: "form"
subtype: "basic"
prompt: "Recycling is a very ____ habit for the environment."
options: ["benefit", "beneficial", "benefited", "beneficially"]
answer: 1
