# ===== item_key 1 · Viết tắt (contractions) trong văn trang trọng =====

---

prompt: "Formal version of: 'The report isn't finished yet.'"
options: ["The report is not finished yet.", "The report isn't finished yet.", "The report ain't finished yet.", "The report's not finished yet."]
