# ===== item_key 1 · Modal verbs theo mức độ chắc chắn (will/should/may/might/could) =====

---
id: "hl_modal_b1"
type: "mcq"
input: "choice"
headword: "hl-modal-certainty"
skill: "form"
subtype: "basic"
prompt: "Social media ____ have negative effects, but this depends heavily on how it is used."
options: ["may", "is", "always", "definitely"]
answer: 0
grammar_article_slug: "hedging-language"
explain: "'may' + V nguyên mẫu là modal hedge ở mức ~50% chắc chắn — phù hợp khi kết quả 'depends on' cách sử dụng."
