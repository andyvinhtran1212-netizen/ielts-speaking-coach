# ===== item_key 1 · Sai giới từ sau động từ cố định (verb + preposition) =====

---
id: "pperr_verb_b1"
type: "mcq"
input: "choice"
headword: "pperr-verb-collocation"
skill: "form"
subtype: "basic"
prompt: "Success in this project depends ____ good teamwork."
options: ["on", "to", "of", "for"]
answer: 0
grammar_article_slug: "preposition-errors"
explain: "'depend on' là collocation cố định. Người Việt hay dịch sát 'phụ thuộc vào' thành 'depend to' — sai."
